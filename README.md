# Hello World Example

This is a hello world example with TypeScript, Rollup, esbuild-based minification, Mocha, ESLint, and Express.

## Scripts

- `npm run lint`: Runs ESLint on `src`.
- `npm run test`: Builds test files with Rollup and runs Mocha.
- `npm run build`: Builds a minified CommonJS bundle into `dist/index.js`.
